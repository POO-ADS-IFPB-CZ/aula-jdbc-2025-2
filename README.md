# Aula JDBC

## Instruções para executar
1. Clonar o repositório
2. Adicionar a biblioteca do Driver JDBC do PostgreSQL no projeto
3. Criar um banco  de dados no Supabase para persistir os dados. O Script está no classroom
4. Renomear o arquivo database.properties.example para database.properties e inserir as informações da sua conexão com o supabase. Utilizar os dados da conexão do tipo Session Pooler
